MPx writer

OPENCP is able to store the sample data directly to mpeg compressed audio streams. At the moment only MP1 and MP2 is support, but this might change in the near future. 
[devpMPx]
  link=devpmpx
  diskManual=off 
  layer=2        
  mode=1        
  freq=44100     
  rate=112000    
  model=2        
  crc=off
  extension=off
  copyright=off
  original=off
  emphasis=0
diskManual
start MPx writing with <ALT>+<s>. Note that this options interferes with the same option in the devpDisk section.
layer
save the audio in either layer 1 or layer 2 format.
mode
set the audio output format:
0 stereo
1 joint stereo
2 dual channel
3 mono
freq
the mixing frequency used to calculate the input sample stream. Valid option are 32000, 44100 and 48000.
rate
set the mpeg audio bitrate/s.
model
set the psychoacustic model. This is used for the internal encoding of the audio data. Valid options are either 1 or 2.
crc
save crc checksums into the audio stream. As this enlarges the file this option should be disabled.
extension
set the extension flag
copyright
set the copyright flag
original
set the original flag
emphasis
set the emphasis flag
