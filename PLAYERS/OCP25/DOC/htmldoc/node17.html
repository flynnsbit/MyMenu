Spectrum Analyzer

The spectrum analyzer uses the fast fourier interpolation to gether information on the audio spectrum used in sample data. The analyzer is started with <a>. This function splits the sound data into many bands of pure sine waves. This is called the spectrum of the sample.

The status bar of this function shows you the range each bar covers and the highest frequency processed (left leftmost bar corresbonds to this frequency). Use <Pgup>,<Pgdown> to change the range. <Home> will set the default value ov 2756Hz.5.10

With <ALT>+<a> the mode of the spectrum function can be toggled. Stereo using two analyzers, mono using only one and a single mode are available. In the single mode the currently selected channel is used as sound source for the analyzer.

<tab> changes the color used for the analyzer.

To get a smooth view the default charset of the graphics adapter is modified. This is working fine with every graphics card under MS-DOS. However Windows95 does not allow the manipulition of characters, so the spectrum analyzer function is not shown properly when running under Windows95. If you see a distorted screen while using the spectrum analyzer function this is normal.
