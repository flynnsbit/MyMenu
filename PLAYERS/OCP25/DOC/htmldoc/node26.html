Wurfel mode

With <w> the wurfel mode is enabled. It's only purpose is to display an animation located in the home directory of OPENCP .5.12 The <tab> key will change the play direction. To save diskspace no animations are included in the distribution of OPENCP. They can be found on the OPENCP homepage (page ). Animations can be gererated with the cpwap program from bitmap files. This program is available on the homepage aswell.
