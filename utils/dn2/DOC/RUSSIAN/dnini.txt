==============================================================================
          Описание параметров запуска Дос Hавигатора (файл DN.INI)
==============================================================================

  * bool - параметр может принимать значения: false/true, 0/1, no/yes
  * str  - строковый параметр
  * long - параметр типа longint

==============================================================================

 [Interface]

 EscForOutputWindow = bool  - Когда установлено значение 0, нажатие Esc в
                              файловой панели аналогично нажатию Alt-F5
                              (Экран Дос, User screen), когда установлено
                              1 - аналогично нажатию Ctrl-O (Окно вывода,
                              Output window)
                              (по умолчанию: 0)

 SkipXLatMenu = bool        - Когда установлено значение 0, во вьюере выбор
                              таблицы перекодировки по Shift-F6 осуществляется
                              из удобного меню, когда установлено 1 - из
                              неудобного, но привычного для некоторых давних
                              пользователей DN файлового диалога
                              (по умолчанию: 0)

 UseEnterInViewer = long    - Когда установлено значение 0, нажатие Enter
                              при просмотре файла не производит никаких
                              действий, когда установлено 1, Enter закрывает
                              окно просмотра, и, наконец, 2 означает, что
                              клавиша Enter должна производить переключение
                              между режимами просмотра текста,
                              шестнадцатеричных кодов и дампа
                              (по умолчанию: 0)

 PreserveMenuPositions = bool - Когда включен, текущие позиции во всех меню
                              сохраняются между запусками
                              (по умолчанию: 0)

 HistoryErrorBeep = bool    - Если включено, пpи нажатии Del в истоpии ввода
                              на "замороженном" элементе выдается звуковой
                              сигнал. В противном случае нажатие Del на таком
                              элементе просто игнорируется
                              (по умолчанию: 1)

 ReuseViewers = long,
 ReuseEditors = long        - первый параметр определяет поведение окон
                              просмотра, второй - поведение окон редактора.
                              Когда установлено значение 1, вместо открытия
                              второго окна для одного и того же файла
                              происходит запрос на активизацию уже
                              существующего окна. При значении 2 активизация
                              существующего окна в таких случаях выполняется
                              без запросов. При нулевом значении всегда
                              открывается новое окно
                              (по умолчанию: 0)

 ReflectCopyDirection = bool - при установленном значении 1 индикатор
                              копирования растет справа налево при
                              копировании с правой панели в левую
                              (по умолчанию: 0)

 VertScrollBarChars = str,
 HorizScrollBarChars = str  - задают символы pисования полос пpокpyтки,
                              состоят из 5 символов: первого и последнего
                              символов, символа полосы, символа бегyнка и
                              символа пyстой полосы. Записывается в виде
                              стpоки из пяти символов
                              (по умолчанию: ..▒■▓ )

 SystemMenuChar = long      - задает символ, используемый для отображения
                              первого подменю главного меню. Символ задается
                              своим номером в таблице ASCII
                              (по умолчанию: 4)

 DriveSelectVCenter = bool  - при установленном значении 1 окно выбора дисков
                              выравнивается по вертикали по центру экрана.
                              (по умолчанию: 0)

 WinManagerSelectNext = bool - при выборе окна по Alt+0 выбирается
                              не текущее окно, а следующее по порядку
                              (по умолчанию: 1)

 CutDriveInfo = bool        - укорачивание описания драйвов
                              в выборе Alt+F1/F2
                              (по умолчанию: 1)

==============================================================================

 [Clock]

 RightAlignClock = bool     - Когда включен, часы всегда находятся вплотную
                              к правому краю экрана
                              (по умолчанию: 0)

 DaysOfWeek = str           - позволяет задать названия дней недели,
                              отображаемые в области часов при нажатой
                              клавише Ctrl, отличные от предусмотренных по
                              умолчанию для данного языка. Если значение
                              этого параметра имеет длину, равную 14 или 21,
                              то из него берется соответственно двух- или
                              трехбуквенное название дня недели. Hазвания дней
                              недели перечисляются без пробелов, начиная с
                              воскресенья, например, SunMonTueWedThuFriSat.
                              Если же параметр имеет другую длину, оставлен
                              пустым или не указан, то используются
                              двухбуквенные названия, определенные в ресурсах
                              для каждого языка в отдельности.
                              (по умолчанию: use language default)

 ShowDayOfWeek = bool       - показывать день недели при удерживании Ctrl
                              (по умолчанию: 1)

 ShowCentury = bool         - показывать четыре цифры для года при
                              удерживании Ctrl
                              (по умолчанию: 1)

 ShowSeconds = bool         - управляет отображением секунд.
                              (по умолчанию: 1)

 BlinkSeparator = bool      - включает/выключает мигание разделителя в часах,
                              когда показ секунд выключен
                              (по умолчанию: 1)


==============================================================================

 [SmartPad]

 LineChar = long            - позволяет задать символ, используемый для
                              разделителя в SmartPad. Символ задается своим
                              номером в таблице ASCII. Псевдографический
                              символ, используемый по умолчанию (-), в
                              некоторых случаях неуместен, так как не
                              поддается перекодировке
                              (по умолчанию: 196)

 InsertDate = bool          - вставлять дату при открытии SmartPad-а
                              (по умолчанию: 1)

==============================================================================

 [Game]

 EnableGame = bool          - установка этого параметра в 0 позволяет
                              отключить возможность запуска встроенной игры
                              (по умолчанию: 1)

==============================================================================

 [Clipboard]

 MaxClipboardSize = long    - максимальный накапливаемый размер Clipboard
                              (по умолчанию: 4096)

 SaveClipboardOnExit = bool - хранить ли Clipboard между запусками
                              (по умолчанию: 1)

==============================================================================

 [Kernel]

 CtrlBreakKill = bool

 SmartWindowsBoxClose = long - Параметр определяет, каким образом DN/2
                               реагирует на попытку закрытия окна консоли
                               системной кнопкой закрытия или из системного
                               меню окна:
                               0 - не реагирует (стандартное поведение для
                                   консольной программы)
                               1 - реагирует как при нажатии Alt-X или выходе
                                   из программы через меню
                               2 - то же, что и 1 , но подтверждения на выход
                                   не выдаётся
                               3 - реагирует как на нажатие Ctrl-Break
                               (по умолчанию: 0)


 StoreVideoMode = long      - 0 означает нигде не сохранять видеорежим,
                              выбранный из меню,
                              1 означает сохранять его как видео режим 1,
                              2 означает сохранять его как видео режим 2
                              (по умолчанию: 0)

 CoрyLimit = long           - указывает ограничение буфера копирования в
                              Windows в килобайтах. Значение 0 означает
                              отсутствие ограничения
                              (по умолчанию: 8192)

 ShowKeyCode = long         - может быть:
                              0: Коды клавиш не показываются
                              1: Показываются пары Scan/ASCII в отдельном
                                 окне
                              2: Показывается Scan-код в верхнем правом углу
                              3: 1 и 2 одновременно
                              (по умолчанию: 0)

 AutoSave = bool            - пересохранять все параметры в DN.INI при
                              запуске
                              (по умолчанию: 1)

 ForceDefaultArchiver = str - может принимать значение одного из
                              поддерживаемых типов архиваторов (см. диалог
                              архивации файлов) или пустой строки. Если задан
                              - архиватором по умолчанию в диалоге
                              архивирования всегда выбирается заданный
                              архиватор

 [Editor]

 AutoBracketPairs = str     - позволяет задать любое количество пар символов,
                              считающихся скобками для функции AutoBrackets
                              (АвтоСкобки). (Примечание: этот параметр не
                              имеет никакого отношения к функции поиска
                              парных скобок)
                              по умолчанию: ()[]{}<>

 WinManagerPosToEdit = bool - при нажатии Alt+0 если находимся в редакторе
                              то курсор встает на следующее окно редактора,
                              а не на любое окно
                              Примечание: Действие флага приоритетно, по
                              сравнению с WinManagerSelectNext
                              (по умолчанию: 1)

 FastSearchDeep = long      - при поиске пар begin/end и т.д. может
                              происходить сбой при наличии комментариев,
                              содержащих ключевые слова (сбивается подсчет
                              пар). Для исключения неограниченного поиска по
                              файлу (очень долго ждать) предлагается
                              установить макс. глубину поиска (около 300 стр.)
                              (по умолчанию: 0) - неограниченный поиск

 DefCodePage = str          - указание кодовой страницы для новых
                              файлов. Может иметь значения Dos, Win, Koi,
                              Auto. При выборе Auto происходит попытка
                              автоматического определения кодировки
                              (по умолчанию: Dos )

 FastBookmark = bool        - отображение полосы быстрого доступа к
                              закладкам внизу окна редактора
                              (по умолчанию: 1)

 ShowBookmarks = bool       - отображать закладки в редакторе
                              (по умолчанию: 1)

 AutoScopeDetect = bool     - автоматическое определение области поиска
                              замены в редакторе: если есть выделение то в
                              Selected text иначе Global
                              (по умолчанию: 1)

 DrawRShift = bool          - переключение между рисованием одинарной и
                              двойной рамкой в редакторе по правому Shift(у)
                              (по умолчанию: 1)

 UnlimitUnindent = bool     - поведение при сдвигании блока влево с
                              помощью Ctrl+K U (сдвиг пока все строки
                              могут сдвигаться или всегда)
                              (по умолчанию: 0)

 F6_DuplicatesLine = bool     - Когда установлено значение 0, в редакторе F6
                              используется для включения и выключения
                              оптимизации пробелов, когда установлено 1 -
                              для удвоения текущей строки
                              (по умолчанию: 0)

 QuickRenameInDialog = bool - значение 1 определяет, что быстрое

 ShowLongName = bool        - показывать длинное имя файла (если есть) в

 UpperCaseSorting = bool          - сортировать имена файлов приведёнными к верхнему
                              (1) или нижнему (0) регистру

 [TaskList]

ShowExePaths = bool         - определяет, показывать исполнимые файлы, из
                              которых был запущен процесс с полным путём (1)
                              или только имя файла (0)
                              (по умолчанию: 0)

KillAllowed = bool          - когда установлено значение 1, имеется
                              возможность убить процесс
                              (по умолчанию: 0)

FilteredList = bool         - когда установлено значение 1, в списке задач

UserTaskFilter = str        - позволяет указать имена исполнимых файлов
                              процессов, которые нужно исключить из списка
                              задач. Имена должны быть разделены символом "#"

 [Language]

 ShowLanguageMenu = bool    - значение 1 означает, что при выборе команды меню
                              "Сменить язык" ("Change language") или нажатии
                              Alt-F12 следует показывать меню, содержащее
                              список всех доступных языков (набор доступных
                              языков определяется путем поиска файлов ресурсов
                              в каталогах DN и %DNDLG%). Это удобно, если в
                              системе установлено более двух языков. Значение
                              0 (по умолчанию) означает, что при выборе
                              указанной команды или нажатии Alt-F12 следует
                              автоматически переключаться по кругу на
                              следующий доступный язык после текущего
                              (по умолчанию: 0)

 HelpLanguageOverride = str - если указано непустое значение, оно считается
                              именем языка для справочной системы. Hапример,
                              при HelpLanguageOverride=english будет
                              использоваться файл справки english.hlp. При
                              пустом значении этого параметра или его
                              отсутствии используется файл справки для
                              текущего (активного) языка. Если для активного
                              языка файл справки отсутствует, используется
                              файл справки для английского языка (English)
                              (по умолчанию: )

 ActiveLanguage = str       - если указано непустое значение, оно считается
                              именем текущего языка. Hапример, при
                              ActiveLanguage=russian будут использоваться
                              файлы ресурсов russian.*. При пустом значении
                              этого параметра или его отсутствии используется
                              переменная окружения DNLNG. Если и этой
                              переменной нет, используется английский язык
                              (English)
